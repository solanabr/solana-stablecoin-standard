# Solana Stablecoin Standard (SSS)

Modular stablecoin framework for Solana with three presets:

- `SSS-1`: minimal role-based stablecoin
- `SSS-2`: compliance add-ons (blacklist, seizure, transfer hook)
- `SSS-3`: privacy module integration (Cloak-compatible relay flow)

See full documentation in `docs/README.md`.
